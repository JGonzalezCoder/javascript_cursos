Ejercicio #2:

Adivina el numero magico, considerando esta linea de codigo con la que puedes obtener un numero aleatorio en Javascript:

let numeroMagico = parseInt(Math.random()*100);

Implementa un programa que solicite al usuario ingresar un numero hasta adivinar el numero magico. El programa debera ademas:
>_ Imprimir "el numero es mayor" si el usuario ingreso un numero menor.
>_ Imprimir "el numero es menor" si el usuario ingreso un numero mayor.
>_ Imprimir "Felicidades, adivinaste el numero" si el usuario ingreso el mismo numero.
>_ Imprimir "Tu opcion esta fuera del rango 0-100" si el usuario ingreso un numero mayor a 100.
>_ Imprimir "Error. Solo puedes ingresar numeros" si el usuario ingreso algo que no sea un numero.

Crea una funcion que reciba la cantidad de horas que has trabajado, y retorne el dinero que has ganado.
